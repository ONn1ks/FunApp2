# FunApp2

Интерактивная браузерная игра, в которой нужно управлять кружком и уворачиваться от охотящихся квадратов. Со временем противников становится больше, а ваш герой — меньше и быстрее, поэтому сложность растёт с каждой секундой.

## Как запустить

1. Склонируйте репозиторий или скачайте архив с файлами.
2. Откройте файл `index.html` в любом современном браузере.

Игра управляется клавишами со стрелками или буквами `W`, `A`, `S`, `D`. На сенсорных устройствах можно удерживать палец на игровом поле — кружок будет бежать к точке касания.
