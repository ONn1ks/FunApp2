# FunApp2

Мини-игра «Убеги от квадратиков», где вы управляете светящимся кружком и
стараетесь продержаться как можно дольше. Со временем врагов становится всё
больше, они ускоряются, а ваш герой уменьшается в размере, но набирает скорость.

## Как играть

1. Откройте `index.html` в браузере.
2. Управляйте кружком стрелками или клавишами WASD.
3. Избегайте столкновений с красными квадратиками — каждое попадание
   заканчивает забег.
4. Наблюдайте за таймером, чтобы побить собственный рекорд выживания.

После проигрыша нажмите «Играть снова» или клавишу Enter, чтобы начать заново.
